# TypeScript example (published package)

This example intentionally installs the published `@kitedb/core` package to validate typings and completions.

## Setup

```bash
cd examples/ts-package
npm install
```

## Run

```bash
npm run dev
```

## Typecheck only

```bash
npm run typecheck
```

If you prefer Bun:

```bash
bun install
bun run dev
```
